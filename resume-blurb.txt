1. PRIMARY VERSION — Projects section entry (6 bullets)
----------------------------------------------------------------

UsedExchange — Static Marketplace Storefront (Personal Project)
Next.js 15 · TypeScript · Tailwind CSS v4 · Zod · GitHub Actions
github.com/WillWYQ/usedExchange · 2026

- Architected a database-free, statically-generated marketplace in
  Next.js 15 / TypeScript where all content is driven from a single
  content/ folder, letting non-technical sellers manage inventory
  without touching application code.

- Designed a distance-tiered pricing engine that resolves buyer
  location entirely client-side via the browser Geolocation API
  (haversine distance, zero server calls), serving location-specific
  prices while keeping visitor coordinates private.

- Engineered a git-free image pipeline that uploads photos to a
  Cloudflare R2 CDN and commits only a lightweight JSON manifest,
  keeping deploy output within free static-hosting limits (vs. ~1 GB
  if 100+ items were bundled) on GitHub Pages.

- Drove a spec-first, AI-assisted workflow with Claude Code, producing
  a ~270 KB internally-consistent design corpus (7 docs, versioned
  v0.2.0 -> v0.9.0) hardened through 10+ AI-led multi-pass consistency
  audits that caught cross-document defects — orphaned schema fields,
  broken build-step ordering, dependency cycles — before any code was
  written.

- Authored a 14-phase, agent-executable implementation plan with
  machine-checkable acceptance criteria (type-check + lint gates),
  enabling an AI coding agent to build and self-verify the system
  phase by phase.

- Designed three portable AI assistant skills (/setup, /update-items,
  /translate-items) that generate site config, item metadata, and
  locale translations from a seller's photos and notes — Markdown
  instruction files requiring no API key or extra dependencies.
